# M6 candidate-2 activation notes

## Paired-incumbent validity coupling

The paired postrepair-incumbent attribution arm is part of the registered
candidate-2 run protocol, not an optional report. Its scores are non-gating and
are not Section 8 acceptance inputs, but the integrity of the Section 7
first-marriage decomposition—including this paired arm—is a run-validity
condition. If that decomposition is invalid, the overall `verdict.status` is
`INVALID`; the independently computed `section_8_acceptance_status` is
preserved separately and must not be presented as the run verdict.
